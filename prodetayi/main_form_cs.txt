`MainForm.cs`, TavlaJules kontrol panelinin ana WinForms ekranidir.

- Koyu temali tek ekran masaustu arayuzu kurar.
- 2026-04-24 guncellemesiyle pencere minimumu kucultuldu, sol ayar paneli scroll destekli hale getirildi ve sag taraf sekmeli tasarima alindi.
- Proje klasoru, GitHub repo, Jules URL, OpenRouter model, OpenRouter API key ve genel hedef alanlarini gosterir.
- Fallback modeller sol ayar panelinde gorunur ve duzenlenebilir hale getirildi.
- Ayarlari `%AppData%\TavlaJules\settings.json` dosyasina kaydeder.
- API key yalnizca proje kokundeki `.env` dosyasina yazilir; bos birakilirsa mevcut `.env` anahtari korunur.
- OpenRouter baglanti testi yapar; primary model calismazsa fallback model zincirine duser.
- Jules CLI version testi yapar ve `jules new --repo ...` ile remote session baslatabilir.
- Jules icin duzenlenebilir gorev promptu alanini, faz planini ve calisma gunlugunu ekranda tutar.
- Sag panelde `Yol Haritasi`, `Ajan`, `Jules Prompt` ve `Gunluk` sekmeleri bulunur; boylece kucuk ekranlarda alanlar birbirini ezmez.
- `Ajan Ayarlari` bolumunde izlenecek Jules session ID, Aiven `ajanlarim` MySQL connection string ve otomatik yeni Jules gorevi acma secenegi bulunur.
- `Completed session sonrasi devam gorevi ac` secenegi, tamamlanan izlenen Jules isinden sonra otomatik tek devam gorevi acilmasini kontrol eder.
- `Completed patch'i uygula ve dogrula` secenegi, completed Jules patch'ini otomatik apply edip build/test dogrulamasi yaptirir.
- `Dogrulanan degisiklikleri commit/push yap` secenegi, dogrulanmis patch'i secret scan sonrasinda GitHub'a gonderen tam otonom modu kontrol eder.
- `Ajan baslat`, her 60 saniyede `TavlaAgentService` uzerinden Jules durumunu kontrol eder ve OpenRouter ajan raporu uretir.
- `AutoStartAgent` ayari true ise pencere acildiktan sonra `Ajan baslat` otomatik tetiklenir; boylece publish EXE yeniden acildiginda dakikalik takip tekrar baslar.
- `Ajan tek tur`, dakika beklemeden tek izleme/analiz turu calistirir.
- `DB test`, `.env` icindeki `AJANLARIM_MYSQL` ile Aiven baglantisini test eder.
- Ajan tek tur tamamlandiginda UI gunlugune SQL rapor yazma sonucunu da basar.
- Ajan yeni Jules session olusturursa UI'daki izlenen session id otomatik yeni id ile guncellenir ve ayarlara kaydedilir.
- Ajan her turda otonom ozetini gunluge yazar; apply/verify/commit/push kararlari dashboard ve SQL eventlerinde ayrica izlenir.
- Proje klasorunu ve GitHub reposunu dis uygulamada acabilir.
