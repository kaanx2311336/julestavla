`ProjectSettings.cs`, TavlaJules icin saklanan temel ayarlari modeller.

- `AgentName`: SQL ve raporlama kimligi; varsayilan `tavlajules`.
- `ProjectFolder`: tavlajules calisma klasoru.
- `GitHubRepo`: Jules'in calisacagi GitHub repo slug'i (`kaanx2311336/julestavla`).
- `JulesUrl`: Jules web adresi.
- `JulesCommand`: lokal Jules CLI komutu (`jules.cmd`).
- `OpenRouterModel`: kontrol icin hedef OpenRouter modeli.
- `OpenRouterFallbackModels`: primary model 503/429 gibi gecici hata verirse denenebilecek modeller; varsayilan artik yalnizca `google/gemma-3n-e2b-it:free`.
- `OpenRouterEndpoint`: OpenRouter chat completions endpoint'i.
- `AgentModel`: dakikalik orkestrator ajanin birincil modeli; varsayilan `openai/gpt-oss-120b:free`.
- `AgentIntervalSeconds`: ajan periyodu; varsayilan 60 saniye.
- `TrackedJulesSessionId`: dakika basi izlenecek Jules session ID.
- `AllowAutoJulesSessions`: true olursa ajan, modelin onerisine gore yeni Jules session acabilir.
- `AutoContinueCompletedSessions`: true olursa izlenen Jules session tamamlandiginda ajan tek seferlik devam session'i acar; varsayilan true.
- `AutoRecoverAwaitingInputSessions`: true olursa izlenen Jules session `Awaiting User Feedback/Input` durumuna duserse ajan bir kez netlestirilmis kurtarma session'i acar; varsayilan true.
- `AutoApplyCompletedSessionPatch`: completed Jules patch'ini otomatik apply etmeyi kontrol eder; varsayilan true.
- `AutoCommitAndPushAppliedChanges`: dogrulanmis patch'i staged secret scan sonrasinda commit/push etmeyi kontrol eder; varsayilan true.
- `AutoRunVerification`: apply sonrasinda `dotnet build` ve `dotnet test` calistirir; UI'da apply secenegiyle birlikte yonetilir.
- `Goal`: projenin genel hedef metni.
