`.env.example`, gizli degerler icin ornek ortam dosyasidir.

- `OPENROUTER_API_KEY` bos birakilir; gercek anahtar `.env` dosyasinda tutulmalidir.
- Varsayilan model `openai/gpt-oss-120b:free`.
- Dakikalik ajan modeli `OPENROUTER_AGENT_MODEL=openai/gpt-oss-120b:free`.
- Fallback modeller `OPENROUTER_FALLBACK_MODELS` ile tutulur; qwen varsayilan zincirden cikarildi.
- OpenRouter endpoint'i `OPENROUTER_API_URL` alaninda tutulur.
- Varsayilan Jules adresi `https://jules.google.com`.
- Varsayilan GitHub repo `kaanx2311336/julestavla`.
- Aiven `ajanlarim` MySQL rapor baglantisi `AJANLARIM_MYSQL` ile tutulur.
- Eski/oyun DB ayri tutulacaksa `TAVLA_ONLINE_MYSQL` sonradan oyun tarafina ayrilabilir.
