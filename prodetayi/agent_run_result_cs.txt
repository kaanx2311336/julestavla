`AgentRunResult.cs`, dakikalik TavlaJules ajaninin tek tur sonucudur.

- Kullanilan model, analiz metni, onerilen sonraki Jules promptu ve rapor yolu tutulur.
- Jules session listesi ve varsa pull ozeti saklanir.
- SQL rapor yazma sonucu `SqlReportMessage` ile UI'a tasinir.
- Otomatik Jules session acildiyse komut sonucu `AutoJulesSessionResult` ile tasinir.
- Yeni takip edilecek Jules session id okunabildiyse `NewJulesSessionId` ile UI ve headless calistirma tarafina tasinir.
