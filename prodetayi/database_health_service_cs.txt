`DatabaseHealthService.cs`, Aiven `ajanlarim` MySQL baglantisini test eder.

- `.env` icindeki `AJANLARIM_MYSQL` connection string ile baglanir.
- URI bicimini veya klasik MySQL connection string bicimini `ConnectionStringService` uzerinden kabul eder.
- `information_schema.tables` uzerinden aktif database icin tablo sayisini okur.
- Baglanti yoksa veya hataliysa secret degerleri loglamadan kisa durum mesajiyla doner.
