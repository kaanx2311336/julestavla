`OpenRouterClient.cs`, OpenRouter chat completions endpoint'ine test istegi atan servistir.

- Endpoint: `https://openrouter.ai/api/v1/chat/completions`.
- Authorization Bearer header'i ile `.env` icindeki API key kullanilir.
- Primary model `OpenRouterModel` uzerinden denenir.
- 400/408/429/5xx gibi gecici veya model kaynakli hatalarda `OpenRouterFallbackModels` sirayla denenir.
- Basarili cevapta kullanilan modeli ve kisa test yanitini UI gunlugune dondurur.
- `CompleteAsync`, dakikalik TavlaJules ajaninin sistem/user promptlarini OpenRouter'a gonderir.
- Model uyumlulugu icin ajan talimatlari ayri `system` mesaji yerine tek `user` mesajinda `TALIMATLAR` + `GOREV VERISI` formatinda gonderilir.
- Ajan icin birincil model `AgentModel` alanidir; varsayilan `openai/gpt-oss-120b:free`.
- Primary ajan modeli gecici hata verirse ayni fallback zinciri kullanilir.
- Model cevabi ```json fenced block ile gelirse temizlenir; SQL ve UI tarafina sade JSON metni tasinir.
